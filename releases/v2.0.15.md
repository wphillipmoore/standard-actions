
# Release 2.0.15 (2026-05-23)

## Bug fixes

- **guard version-divergence fetch against credential race**
Skip the git fetch origin main when origin/main is already available locally, avoiding intermittent credential-helper failures in container jobs. Ref #561

- **skip Semgrep language ruleset for unsupported languages**
Replace the runtime curl check against the Semgrep registry with a static allowlist of languages that have a known p/<language> ruleset. Languages like shell that have no Semgrep ruleset are now silently skipped instead of emitting a noisy ::notice:: annotation. Ref #559


## Chores

- **bump version to 2.0.15**
- **add missing container-tag and container-suffix parameters**
Add container-tag: 'latest' to quality, security, and version jobs. Add container-suffix: base to security and version jobs.

Ref vergil-project/vergil-tooling#1015, vergil-project/vergil-tooling#1013, vergil-project/vergil-tooling#976


## Features

- **fail CI on CodeQL findings at warning level or above**
Enforce zero-tolerance security policy by checking SARIF output after CodeQL analysis and exiting non-zero when any warning- or error-level findings exist. Previously these appeared only as PR annotations and did not block merges. Ref #556
