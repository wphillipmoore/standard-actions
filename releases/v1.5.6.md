
# Release 1.5.6 (2026-05-06)

## Bug fixes

- **skip uv tool install when consumer repo is standard-tooling**
When CI runs against a standard-tooling PR, the setup action was installing the released version (from the tag in standard-tooling.toml) instead of the PR's code. Detect this case by checking pyproject.toml for the standard-tooling package name and skip the uv tool install so uv sync --group dev --frozen provides the PR's code on PATH. Closes #354
