
# Release 1.5.5 (2026-05-06)

## Bug fixes

- **install dev dependencies in lint and typecheck jobs**
The lint and typecheck jobs install standard-tooling but never install the calling repo's dev dependencies. For Python repos, ruff and mypy are not on PATH when st-validate invokes them. Add a conditional step to run uv sync --group dev --frozen for Python repos before running st-validate. Closes #349

- **install dev dependencies in test and audit reusable workflows**
Same fix as ci-quality.yml: the ci-test.yml and ci-audit.yml reusable workflows need dev dependencies installed for Python repos so that pytest, uv lock --check, and other dev tools are available when st-validate invokes them.

- **rename step from Install dev dependencies to Install dependencies**
uv sync --group dev installs all dependencies (main + dev), not just dev. The step name was misleading.
