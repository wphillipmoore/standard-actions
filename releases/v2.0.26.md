
# Release 2.0.26 (2026-05-28)

## Bug fixes

- **separate language and container-suffix inputs in cd-release.yml**
Add dedicated container-suffix input so non-language projects no longer misuse language: base. The container image line falls back through container-suffix, language, then 'base'. The validate-inputs action now treats language as optional and only passes it to the CLI when non-empty. Ref #653
