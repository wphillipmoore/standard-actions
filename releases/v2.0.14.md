
# Release 2.0.14 (2026-05-22)

## Bug fixes

- **use app token for github-config audit**
The audit calls the Actions permissions API which requires
administration:read — a scope not available on GITHUB_TOKEN.
Switch to a GitHub App token, matching the pattern used by
the CD release workflows.

Callers must now pass APP_CLIENT_ID and APP_PRIVATE_KEY secrets.
