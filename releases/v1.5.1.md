## Features

- **add p/ci base ruleset and auto-detect p/dockerfile, p/github-actions**
Add p/ci (147 CI pipeline security rules) to the base Semgrep config that runs for every scan. Auto-detect Dockerfiles and GitHub Actions workflow files in the repo and enable p/dockerfile and p/github-actions rulesets accordingly, following the same file-presence detection pattern used by ci-quality's common job. Consuming repos get expanded coverage automatically — no configuration changes needed. Ref #303
