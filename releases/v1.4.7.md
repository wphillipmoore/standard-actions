
# Release 1.4.7 (2026-05-01)

## Refactoring

- **replace pip install with uv tool install and remove guard patterns**
Standardize on uv for tool installation across all composite actions, removing runtime detection guards that second-guess what should be guaranteed at setup time. standards-compliance: remove command -v guard and st-config.toml fallback, replace pip install with uv tool install. docs-deploy: remove command -v mike detection and conditional setup-python, replace pip install with uv tool install mike --with mkdocs-material, use uv run --with pyyaml for nav patching. security/semgrep: replace pip install with uv tool install. Ref #287

- **remove runtime tool installs now that dev-base image provides them**
Now that mike, mkdocs-material, pyyaml, and semgrep are baked into the dev-base container image, remove the runtime install steps from composite actions and add container directives to workflows that were running on bare runners. docs-deploy: remove uv tool install mike step and uv run --with pyyaml wrapper. security/semgrep: remove uv tool install semgrep step. docs.yml: add container directive so mike is available. ci-security.yml: add container directive to semgrep job. Ref #287
