
# Release 2.0.25 (2026-05-28)

## Bug fixes

- **revert to Docker-based approach for bare-runner compatibility**
The vrg-trivy-scan rewrite (commits 3e3f333b through c4956a80) broke CD pipelines on bare ubuntu-latest runners where the trivy binary is not installed. Restore the Docker-based approach that brings its own Trivy via aquasec/trivy:0.70.0, with injection-safe env: blocks (from 7290af2d).

Also remove the container directive from the ci-security trivy job since Docker-based Trivy runs on the host and does not need a container image.
