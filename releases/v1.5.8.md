
# Release 1.5.8 (2026-05-06)

## Bug fixes

- **prepend .venv/bin to PATH before st-validate calls**
GitHub Actions overrides the container WORKDIR to /__w/<repo>/<repo>,
so the Docker image's /workspace/.venv/bin PATH entry never resolves.
Prepend $GITHUB_WORKSPACE/.venv/bin to PATH in every st-validate run
block so that st-validate itself and dev tools it invokes (ruff, mypy,
pytest, pip-audit) are findable.  Harmless no-op for non-Python repos.
