
# Release 2.0.16 (2026-05-23)

## Bug fixes

- **add missing contents:read permission to security scanner jobs**
The codeql, trivy, and semgrep jobs overrode workflow-level permissions with only security-events:write, implicitly setting contents to none. This caused intermittent checkout failures when the GITHUB_TOKEN lacked the scope needed to fetch repository contents. Ref #568
