
# Release 1.5.17 (2026-05-09)

## Bug fixes

- **keep release job key until check name registry updates (#383)**
- **use system Python to avoid venv PATH shadowing (#418)**
The inline Python scripts in docs-deploy and cd-docs resolve python3 via PATH. When the consumer repo is standard-tooling itself, the setup action creates a project venv and prepends it to PATH — shadowing the system Python that has pyyaml and other packages baked into dev-base. Use the absolute path /usr/local/bin/python3 instead.
