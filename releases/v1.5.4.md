
# Release 1.5.4 (2026-05-06)

## Bug fixes

- **use relative action refs in CI workflows, freeze to tagged refs at publish time**
Convert @develop remote refs in CI gate workflows to ./actions/ relative
paths so standard-actions tests its own action code during development.
Update the publish.yml freeze step to rewrite ./actions/ refs to
fully-qualified tagged refs at release time, and add a validation step
to assert no unfrozen refs survive in the tagged commit.
