
# Release 1.5.10 (2026-05-06)

## Bug fixes

- **run uv sync in standard-tooling self-install path**
When the composite action detects the consumer repo is standard-tooling itself, run uv sync --frozen --group dev to install CLI tools from source into .venv/bin/ instead of just skipping installation entirely. Ref #373


## Features

- **add container-tag input, make common job container overridable**
The common job now uses container-suffix (defaulting to base) and container-tag (defaulting to latest) instead of hardcoding dev-base:latest. This lets callers like standard-tooling provide a Python-capable container for the self-install case. Ref #373

- **add container-suffix and container-tag inputs for standards job**
The standards job now uses container-suffix (defaulting to base) and container-tag (defaulting to latest) instead of hardcoding dev-base:latest. This lets callers like standard-tooling provide a Python-capable container. Ref #373

- **add container-suffix and container-tag inputs for version-bump job**
The version-bump job now uses container-suffix (defaulting to base) and container-tag (defaulting to latest) instead of hardcoding dev-base:latest. This lets callers like standard-tooling provide a Python-capable container. Ref #373
